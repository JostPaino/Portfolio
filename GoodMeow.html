Good Meow-ning App Design with Complimentary Landing Page

Project Overiew


The Product

Good Meow-ning takes pride in being a one-stop-shop for all things cat and coffee. They need a new home page and a mobile app that allows customers to place orders quickly and easily.

The Product Duration

1 week

The problem:

Their current consumer base would prefer a mobile application because they can't use a PC when they are on the go.

The goal:

Create an application and landing page that will provide users with the ability to order quickly and effectively.

My role:

UX designer designed the mobile application and complimentary website from start to finish.

Responsibilities:

Conducting interviews, paper and digital wireframing, low and high-fidelity prototyping, conducting usability studies, accounting for accessibility, and, iterating on designs.

Understanding the User

User research: summary

I used Good Meow-ning's data to create interview questions, which I then used to conduct user interviews. The majority of those surveyed indicated they'd like to be able to place orders fast while on the go.
